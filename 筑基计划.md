## 第一部分

使用Python构建CLI式可读取/写入的多条目数据库  
要点：

* 读取写入方法实现  
* 序列化多文件存储  
* 接口化  
* 数据格式化  
* 窗口维持

**读取写入方法实现**  
结构：包括索引json和数据库json  
索引内部只含id 名称 别名  
数据库含id 详细介绍 标签 图片  
注意！在写入时候需要拆分至索引和数据库且同时写入  
**读取**  
匹配时需要用包含 也就是 in 条件  
同时我们可以遍历索引然后把名字 别名 标签 使用加号拼起来然后检查整个这个字符串  
Python 自带 json 库。  
​读取：data \= json.load(open('data.json')) —— 一行搞定。  
​写入：json.dump(data, open('data.json')) —— 一行搞定。

## 第三部分

工程和扩展化，包括两个阶段

​路由： React Router (怎么实现页面跳转)。  
​第二阶段：工程化与生态  
​TypeScript： 给JS加上类型约束。日本非常看重TS，因为他们喜欢严谨的代码。  
​框架： Next.js (服务端渲染) 或 继续深挖 Vite 生态。
