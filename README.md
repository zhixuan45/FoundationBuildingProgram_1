# 赛马娘/明日方舟 本地wiki

这是一个结合了赛马娘和明日方舟元素的本地wiki系统，旨在创建一个可搜索的角色数据库，支持多种过滤和展示功能。

## 项目概述

该项目是"筑基计划"的一部分，旨在增强技术基础并挑战极限。项目使用前后端分离架构，包含数据库、后端API和前端界面，能够展示和管理角色数据。

## 技术栈

* **数据库**：使用JSON文件存储角色数据（ID、名称、别名、详细信息、标签、图片路径等）
* **后端**：使用Python实现API接口，向前端提供数据
* **前端**：使用HTML/CSS/JavaScript构建用户界面，展示卡片列表和搜索功能

## 功能特性

* **角色数据管理**：存储角色的基本信息、详细介绍、标签和图片
* **搜索功能**：支持按名称、别名、标签进行搜索
* **分类筛选**：可以按类别（如"草地/泥地"或"近卫/术士"）进行筛选
* **详情查看**：点击角色可以查看大图和详细信息

## 项目结构

* **jsondatabase.py**：Python后端，处理数据的读取和写入
* **script.js**：前端JavaScript逻辑
* **index.html**：主页面结构
* **style.css**：样式文件
* **index_file.json**：索引文件，包含ID、名称和别名
* **details_file.json**：详细数据文件，包含角色的详细介绍、标签和图片

## 项目目标

最终实现一个功能完整的本地wiki系统，具备：
* 角色卡片列表展示
* 搜索和筛选功能
* 详情页浏览
